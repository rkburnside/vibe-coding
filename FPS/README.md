# Arena FPS

A simple 3D 1st person shooter game in an arena against 1 computer opponent.

## Controls

- **Arrow Up / Down**: Move Forward / Backward
- **Arrow Left / Right**: Turn Left / Right
- **Space**: Shoot
- **R**: Restart Game

## Technical Details

- Built with [Three.js](https://threejs.org/).
- Features a 3D arena with lighting, pillars for cover, and a fog effect.
- The computer opponent has a basic AI that faces the player, strafes, and shoots.
- Includes a HUD (Heads-Up Display) with health bars for both the player and the enemy.

## How to Run

Simply open `index.html` in any modern web browser.
